# Gianfranco Mongiello — Portafolio Personal

> SAP ABAP Developer · Ing. Sistemas UNIMET · Venezuela 🇻🇪

**Live:** https://portafolio-omega-ecru.vercel.app

---

## Stack

HTML5 · CSS3 · JavaScript vanilla · Chart.js · Vercel

Sin frameworks. Sin dependencias de build. Un solo `git push` y Vercel despliega en 10 segundos.

---

## Features

- Partículas animadas en canvas con conexiones entre nodos
- Efecto glitch en el nombre hero (CSS clip-path animations)
- Typewriter con 5 roles rotativos
- Cursor personalizado con dot + ring (requestAnimationFrame)
- Scroll progress bar con gradiente
- Dark / light mode toggle
- Skill cards con efecto tilt 3D en mousemove
- Barras de nivel animadas con IntersectionObserver
- Heatmap de actividad estilo GitHub (26 semanas)
- Stats de GitHub en tiempo real via API pública
- Modales de proyectos con stack, features y links a repos
- Radar chart de skills + bar chart de la Quiniela WC 2026
- Formulario de contacto con fallback a WhatsApp
- CV descargable en PDF (bilingüe ES/EN)
- Easter egg Matrix (`</>` esquina inferior derecha)
- Favicon SVG inline con iniciales GM
- Responsive mobile — grid de 12 columnas
- Iconos SVG con `currentColor` — adaptativos a dark/light
